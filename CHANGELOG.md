# История изменений

## 1.1.9 - 2026-04-24

- Убраны полосы прокрутки из виджета показателей.
- Исправлена пустая карточка при отсутствии выбранных показателей.
- Карточка пустого состояния теперь занимает виджет целиком без лишнего overflow.

## 1.1.8 - 2026-04-24

- Виджет показателей оформлен карточками в стиле погодных плиток.
- В каждой карточке виджета отображаются название датчика, крупное значение и тип показателя.
- Для температуры, влажности, давления, электрики, батареи, частиц и освещенности добавлены фоновые пиктограммы.

## 1.1.7 - 2026-04-24

- Добавлено перетаскивание плиток мышью с сохранением порядка.
- Виджет показателей Windows стал плавающим: его можно перетаскивать и менять размер.
- Положение и размер виджета показателей сохраняются в настройках приложения.

## 1.1.6 - 2026-04-24

- Возвращены небольшие отступы между плитками в плотной раскладке.

## 1.1.5 - 2026-04-24

- Убраны зазоры между плитками в плотной раскладке.
- Исправлено принудительное включение светлой темы.
- Убрана постоянная фиолетовая подсветка закрепленных показателей в плитках.

## 1.1.4 - 2026-04-24

- Убраны управляющие кнопки из плиток устройств: в плитках остаются только показатели датчиков и переключатель питания.
- Добавлена кнопка автоподбора размеров всех плиток: свернуть до базового вида или раскрыть под доступные показатели.
- Добавлены темы оформления: системная, светлая и темная.

## 1.1.3 - 2026-04-24

- Добавлено правило ветвления: `test` используется для тестовых сборок вида `1.1.1`, `1.1.2`, `1.1.12`.
- Добавлено правило ветвления: `release` используется только для стабильных релизов вида `1.1`, `1.2`.
- Стабильные версии продвигаются в `release` только после проверки и отсутствия известных ошибок.

## 1.1.2 - 2026-04-24

- Добавлена Masonry-перегруппировка плиток без пустых мест между строками.
- Плитки теперь перемещаются относительно друг друга при изменении размера.
- Компактные элементы управления в плитках больше не разрывают карточку сложными наборами режимов.
- Сетка плиток пересчитывается при изменении размера окна и при растяжении плиток.

## 1.1.1 - 2026-04-24

- Добавлен журнал изменений на русском языке.
- Подготовлена публикация истории проекта в GitHub: `git@github.com:AlexZet098/YandexSmartHomeForWindows.git`.
- Принято правило: все дальнейшие доработки фиксируются отдельной версией и русским сообщением коммита.

## 1.1.0 - 2026-04-24

- Добавлена штатная авторизация через Яндекс ID.
- Подключено получение устройств, комнат, сценариев и управление устройствами через Yandex Smart Home API.
- Добавлен светлый интерфейс в стиле официального приложения Яндекс Умный дом.
- Добавлены официальные изображения типов устройств.
- Реализованы адаптивные плитки устройств с плавным изменением размера.
- Добавлены переключатели питания, параметры устройств, управление возможностями и локальная история показаний с графиком.
- Добавлено закрепление выбранных показателей датчиков для отображения в Windows.
- Добавлен виджет показателей рядом с панелью задач Windows.
- Добавлен расчет суммарной мощности устройств с именем `Розетка`.
- Добавлена работа в фоне, tray-меню и автозапуск вместе с Windows.
